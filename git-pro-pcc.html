Co je potřeba vědět ke GITu na PPC?

K čemu to slouží?

Git je obecně nístrojem pro zálohování aktuální verze adresáře. Již není potřeba pojmenovávat milion souborů názvy hw01, hw01_final, hw01_opravdu_final atd. Pomoci nástroje Git je možné vrátit se k nějaké verzi souborů a stavů adresáře. Také je možné nahrát tento adresář na vzdálený repozitář, kterým je v našém případě GitLab.

Začináme

Nejdřive je potřeba stáhnout git. Otevřeme bash, přejdeme do potřebného adresáře pomoci cd a založíme nový přikazem mkdir. Následujícímí přikazy nainstalujeme git.

 # Pro Ubuntu
 
 apt-get update
 apt-get install git
 
 # Pro Fedoru
 
 dnf install git
Dále je potřeba založit adresář. Je to možné buď lokálně přikazem git init nebo založením adresáře pomoci GUI na GitLabu a následným přikazem git clone adresar. Vypadá to následně (stačí z přikazové řádky prohlížeče okopírovat odkaz): 
 git clone https://gitlab.fit.cvut.cz/lytvyden/webpage
Toto je potřeba udělat pouze jednou. Pokud potom náhodou budete pracovat na více počitačích, aktuální verzi adresáře je možné stáhnout pomoci git pull.
